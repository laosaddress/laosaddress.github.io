linux Centos7离线安装nginx

  1. 首页
  2. 文章分类
  3. 正文

离线安装nginx，摸索了一天的成果，怕自己忘记，也分享给大家，有问题请指出。

安装环境

	操作系统：centos7 	联网状态：无互联网-内网

准备工作

			一、准备环境包gcc gcc-c++ 			二、准备准备离线安装包 					openssl 					zlib 					pcre 					nginx

下载安装包

推荐大家下载包的网站：
http://rpmfind.net/linux/rpm2html/search.php
https://centos.pkgs.org/
这两个网站都可以下载所需要的安装包，不同的就是版本差异，大家各取所取。

安装

开始安装

  1. 把包放在一个文件夹中，然后运行命令：rpm -Uvh *.rpm --nodeps --force；
  2. 安装pcre：
    tar -zxvf pcre-8.**.tar.gz
    ./configure
    make
    make install
    大家根据自己的版本修改命令；
  3. 安装openssl：
    tar -zxvf openssl-1.0.**.tar.gz
    ./configure
    make
    make install
    大家根据自己的版本修改命令
    注意，这个包安装的时候一定要进文件夹看清楚文件的名称大小写，如果上面命令的不行，就换config；
  4. 安装zlib：
    tar -zxvf zlib-1.2.**.tar.gz
    ./configure
    make
    make install
    大家根据自己的版本修改命令，同上；
  5. 安装nginx：
    tar -zxvf nginx-1.15.**.tar.gz
    ./configure
    make
    make install
    大家根据自己的版本修改命令；
  6. 执行 nginx命令：
    cd /usr/local/nginx/sbin
    ./nginx；
  7. 浏览器已可访问nginx，打开浏览器访问此机器的 IP，如果浏览器出现 Welcome to nginx! 则表示 Nginx 安装并运行成功。；
  8. 部分命令如下：
    重启：
    /usr/local/nginx/sbin/nginx –s reload
    停止：
    /usr/local/nginx/sbin/nginx –s stop
    测试配置文件是否正常：
    /usr/local/nginx/sbin/nginx –t
    强制关闭：
    pkill nginx

完成
