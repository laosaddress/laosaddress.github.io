使用VS 新建C++项目

  1. 首页
  2. 文章分类
  3. 正文

在Visual Studio 中新建一个C++的项目：
1.打开VS ，看到下面图片的界面后双击 新建项目，图片中红色对勾标部分（或者点击界面左上角的的文件—新建—项目也行）0；

2.然后在下图界面中， 选择 空项目 选项，红对勾部分，记得在 名称 一栏填上项目名称，然后点击确定；

3.接着再鼠标右键 源文件—添加—-新建项， 选择C++（.cpp）文件，填上名称，然后点击添加，这样一个C++项目就创建完成了。


下面测试一下，是否创建成功：

#include<iostream> using namespace std;  int main() { int v1;     cin>>v1;     cout<<v1<<endl;     system("pause");     return 0;  }
